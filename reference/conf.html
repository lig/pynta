Configuration¶

It is likely that settings module is placed in the project root directory and main pynta binary is started from that directory.

class pynta.conf.Settings(settings_module)[source]¶

Proxy for project settings module.

Lets access project settings as pynta.conf.settings module.

class pynta.conf.provider.SettingsConsumer(settings, *args, **kwargs)[source]¶

Base for all abstract base classes to be used as settings sections.

class pynta.conf.provider.SettingsProvider[source]¶

Provider for various settings sections. Allows embedding of section based settings slices depending on templates, storage or any other settings depended classes connected to app class.
